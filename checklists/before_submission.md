# 📋 Pre-Submission Checklist

Final review before sending to a journal. Go through every item.

---

## 📄 Manuscript Quality

### Structure
- [ ] Abstract: 150-200 words, contains Question → Method → Finding → Contribution
- [ ] Introduction: Hook → Gap → This Paper → Preview of Findings → Contributions → Roadmap
- [ ] Literature Review: Organized by theme (not chronologically), ends with "This paper differs by..."
- [ ] Data Section: Source, sample construction, variable definitions table, descriptive statistics
- [ ] Empirical Strategy: Model equation, identification assumptions stated explicitly, parallel trends explained
- [ ] Results: Baseline → Robustness → Heterogeneity → Mechanism, each with economic interpretation
- [ ] Conclusion: Summary → Contributions → Policy Implications → Limitations → Future Research
- [ ] All sections present and in logical order

### Writing Quality
- [ ] First sentence of each paragraph is the topic sentence
- [ ] No "throat-clearing" sentences ("It is worth noting that...")
- [ ] Causal language only where identification strategy supports it
- [ ] Active voice preferred ("We find..." not "It was found...")
- [ ] Consistent tense: literature (present), methods (past), conclusions (present)
- [ ] Run spell-check and grammar check (Grammarly or equivalent)

---

## 📊 Tables & Figures

- [ ] All tables have clear titles and column headers
- [ ] Dependent variable stated in column header or table title
- [ ] Standard errors in parentheses with clustering level noted
- [ ] Significance stars defined: *** p<0.01, ** p<0.05, * p<0.1
- [ ] N (observations), R², and FE specification reported in every regression table
- [ ] Descriptive statistics table includes: N, Mean, SD, Min, Max
- [ ] All figures have axis labels, legends, and source notes
- [ ] Event study plot has clear pre/post markers and confidence intervals
- [ ] Tables and figures numbered sequentially and referenced in text
- [ ] All numbers in text match numbers in tables

---

## 🔬 Empirical Rigor

- [ ] **At least 5 robustness checks** completed and reported
- [ ] **Endogeneity addressed**: Placebo test, pre-trends, or IV validity tests
- [ ] **Economic significance** discussed (not just statistical significance)
- [ ] **Effect size** compared to dependent variable mean
- [ ] **Mechanisms** explored (at least 1-2 channels)
- [ ] **Heterogeneity** by at least 2 dimensions
- [ ] **Parallel trends** tested and plotted (if DID)
- [ ] All key assumptions explicitly stated

---

## 📚 References

- [ ] All cited papers appear in reference list
- [ ] All references in the list are cited in text
- [ ] Citation format consistent (Author, Year) throughout
- [ ] Recent papers included (within last 3-5 years)
- [ ] Seminal papers in your field cited
- [ ] No fabricated/hallucinated references (verify each one)

---

## 💾 Replication Package

- [ ] All code runs from a single master script (one-click replication)
- [ ] Code is well-commented with headers explaining each section
- [ ] Raw data included or access instructions provided
- [ ] Data dictionary / codebook included
- [ ] README.md in replication package explains folder structure and how to run
- [ ] Random seeds set for any stochastic processes
- [ ] All file paths are relative (no hardcoded absolute paths except root)
- [ ] GitHub repository created and linked in paper appendix

---

## ✉️ Submission Logistics

- [ ] Target journal identified (check scope, recent publications)
- [ ] Formatting matches journal requirements (margins, font, spacing)
- [ ] Cover letter drafted (1 page: motivation, contribution, fit with journal)
- [ ] Co-author approval obtained
- [ ] Conflict of interest disclosed
- [ ] Word/page count within journal limits
- [ ] Appendix organized (online supplement if needed)
- [ ] SSRN/arXiv pre-print uploaded (optional but recommended)
